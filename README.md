## Usage

This project using node >= 14.19.2 & yarn

### Installation

```bash
git clone https://github.com/leesharon/codebox-backend.git
yarn install
```

#### Development

```bash
yarn dev
```

You can start editing the server by modifying `server.js`. The server auto-updates as you edit the file.
